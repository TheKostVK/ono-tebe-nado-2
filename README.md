# Проект: Оно тебе надо 2

## Описание
Вёрстка и клиентская логика аукциона по ТЗ.  
Применены технологии: HTML, SCSS, TypeScript, Webpack.  
Для построения интерфейса используются Grid и Flexbox.

## Демо
Опубликованная версия: https://thekostvk.github.io/ono-tebe-nado-2/

## Особенности
- Простой и понятный код
- Компонентная структура проекта
- БЭМ
- Grid
- Flexbox
- TypeScript
- Сборка через Webpack

## Требования
- Node.js (версия 14 или выше)
- npm (актуальная версия)

## Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run start
```

### Сборка для продакшена
```bash
npm run build
```
